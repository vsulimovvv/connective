  • Длительность: 5 дней
  • Язык: русский, английский
  • Требования к слушателям: опыт работы с Windows приложениями

  • Обзор курса:

    Курс разработан для конструкторов, начинающих работать в NX CAD и желающих повысить эффективность и качество своей работы. Полученные знания будут использованы для решения реальных задач предприятия. Курс предполагает достаточное время для выполнения практических примеров с целью лучшего освоения материала.

  • Назначение курса:

    Целью курса является обучение пользователей навыкам работы в NX. Слушатели ознакомятся с интерфейсом пользователя, научатся создавать параметрические и непараметрические кривые, твердотельную геометрию, получат представление о модулях проектирования объектов свободной формы, оформления чертежей и работы со сборками. Знание материала, излагаемого в данном курсе, является необходимым для дальнейшего углубленного изучения NX.

  • Содержание курса:

    • • Интерфейс пользователя NX.
    • • Работа с системами координат.
    • • Работа с эскизами.
    • • Параметризация.
    • • Ссылочная геометрия.
    • • Базовые элементы построения твердотельной геометрии.
    • • Структура детали.
    • • Использование эскизов.
    • • Обрезка тел.
    • • Опции базовых элементов построения.
    • • Создание отверстий.
    • • Создание оболочек.
    • • Ассоциативное копирование элементов.
    • • Операции над ребрами.
    • • Введение в модуль работы со сборками.
    • • Добавление и ограничение компонентов.
    • • Введение в «Черчение».
    • • Введение в «Синхронную технологию».
